# Equipment Rental API (Spring Boot + Angular)

## Description
A full-stack web application built using Spring Boot and Angular to manage hotel reservations and room availability.

## Features
- REST API for managing reservations and rooms
- Multi-language support (English and French)
- Time zone conversion (ET, MT, UTC)
- H2 in-memory database integration
- Frontend built with Angular

## Tech Stack
- Java (Spring Boot)
- Angular
- REST API
- Maven
- H2 Database

### Backend

./mvnw spring-boot:run
